# Instalar as dependências

npm install

## Verificar se há atualizações disponíveis

npm run deps:check

## Atualizar todas as dependências

npm run deps:update

## Atualizar apenas o discord.js

npm run deps:update discord.js

## Verificar vulnerabilidades

npm run deps:audit

## Fazer limpeza completa

npm run deps:clean

## Verificar segurança das versões

npm run security-check

## Iniciar o bot (já faz verificação automática)

npm start / node . / node index.js
